# MindBridge

### A Web Platform for Mental Well-being and Community Support

MindBridge is a full-stack web application designed to promote mental well-being through mood tracking, personal journaling, and supportive communities. The platform provides users with a safe and interactive space to monitor their emotional health, reflect on their thoughts, and connect with like-minded individuals through support circles.

The project aims to combine self-reflection tools with community engagement, creating an environment where users can better understand their emotions while building meaningful connections with others.

## Key Features

* Mood Tracking and Mood History
* Personal Journal with Saved Entries
* Community Support Circles
* User Profile Management
* User Authentication (Login & Signup)
* Responsive and User-Friendly Interface
